# AutoBrowser MCP
AutoBrowser MCP is a Model Context Provider (MCP) server that allows AI applications to control your browser.

Welcome to the **Public Issue Tracker** for AutoBrowser!

Since the main repository is private, this repository exists to allow users to report bugs, request features, and provide feedback. Please follow the guidelines below when filing an issue.

## 🚀 How to Report an Issue

1. **Check for duplicates** – Before creating a new issue, please search existing issues to see if your concern has already been reported.
2. **Use clear and descriptive titles** – This helps maintainers quickly understand the issue.
3. **Provide a detailed description** – Include relevant details like:
   - Steps to reproduce the issue
   - Expected and actual behavior
   - Screenshots or logs (if applicable)
4. **Specify environment details** (if applicable) – Include OS, browser, and version numbers.

## 📝 How to Request a Feature

If you have an idea for a new feature:
1. **Check existing feature requests** to see if a similar request has been made.
2. **Explain the problem** – Describe the issue that your feature request aims to solve.
3. **Suggest a solution** – If possible, provide details on how the feature could work.
